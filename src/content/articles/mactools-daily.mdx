這篇整理了我在 Mac 上日常使用的工具，從輸入法、效率工具到系統維護都有。這些工具之間會互相搭配使用，例如 Raycast 作為中央控制台串接其他工具、Ice 跟 BetterDisplay 一起管理螢幕顯示等等。

## 打字與輸入

### Typeless

<Image src={typeless} alt="Typeless" />

一款語音輸入工具，但不只是單純的語音轉文字而已。

#### 智慧修飾

Typeless 在轉譯語音之後，還會自動幫你去除贅字跟語氣詞，把口語化的內容修飾成更通順的書面文字。不用再花時間回頭刪掉那些「嗯」「然後」「就是」之類的東西。

#### 內容整理

更強大的是，它可以把一大段比較零散、發散的想法，整理成有條理、好閱讀的格式。最後產出的文案不會看起來像是隨口說說的逐字稿，而是一段結構清晰、可以直接拿去用的內容。

#### 全平臺支援

比較特別的是，Typeless 幾乎支援所有主流平臺——Android、iPhone、Mac 都能用。不管在哪個裝置上，都可以隨時用語音輸入來快速產出文字。

<span class="pricing-tag freemium">免費試用，後續訂閱制</span>
<a href="https://www.typeless.com/zh-tw/downloads" target="_blank" class="official-link">官方網站</a>

### SayIt

<Image src={sayit} alt="SayIt" />

同樣是語音輸入工具，我主要把它當作 Typeless 免費額度用盡時的備援方案。

#### API 計價模式

跟 Typeless 一定要繳訂閱費不同，SayIt 可以直接用 API 計價的方式付費，用多少付多少。對於使用量不那麼大、或是想控制成本的人來說，彈性比較高。

#### 與 Typeless 的差異

在動效跟操作體驗上，SayIt 已經有 Typeless 七、八成的表現，日常使用沒什麼問題。不過比較明顯的差距在於：處理非常長的語音文本時表現會弱一些，而且它比較不會去讀取你目前螢幕上的內容來輔助判斷，所以在某些情境下精準度會稍微差一點。

<span class="pricing-tag">程式免費，API 用量計費</span>
<a href="https://jackle.pro/tools/sayIt" target="_blank" class="official-link">官方網站</a>

### CapsLockNoDelay

<Image src={capslocknodelay} alt="CapsLockNoDelay" />

一個很小的工具，專門修正 Mac 上按 CapsLock 切換輸入語言時會有的延遲問題。裝了之後切換變得即時反應，打字節奏不會被打斷。如果你也覺得 Mac 原生的切換延遲很煩，這個工具裝上去就對了。

<span class="pricing-tag">開源免費</span>
<a href="https://github.com/gkpln3/CapsLockNoDelay" target="_blank" class="official-link">GitHub</a>

### 唯音輸入法

<Image src={vchewing} alt="唯音輸入法" />

一款開源的注音輸入法，整體的輸入體驗跟選字邏輯都蠻不錯的，打起來很流暢。不過偶爾會遇到一些小 Bug，穩定性上還有進步的空間。

#### 高度自訂化

支援就地加詞、刪詞、調整詞頻等功能，還可以透過 JSON 匯入匯出偏好設定，讓你完全掌控自己的詞庫跟輸入行為。

#### 打字行為學習

內建打字行為記憶模組（POM），會根據你的打字習慣自動調整選字結果，用越久越懂你的選字偏好。

#### 多種鍵盤排列

除了標準注音之外，也支援漢語拼音等多種鍵盤配置，甚至可以搭配行列等其他輸入方案使用。

<span class="pricing-tag">開源免費</span>
<a href="https://vchewing.github.io" target="_blank" class="official-link">官方網站</a>

### 小麥輸入法（少用）

<Image src={mcbopomofo} alt="小麥輸入法" />

同樣是開源的注音輸入法，走的是輕巧簡潔的路線，安裝容易、操作直覺。

#### 智慧選字

使用自行開發的 Gramambular2 斷字引擎，透過 n-gram 語言模型來做選字判斷，準確度蠻高的。

#### 跨平臺支援

除了 macOS 之外，還有 Linux 版（fcitx5）跟 Chrome OS 版，如果你同時有在用 Chromebook 或 Linux 也能無縫銜接。

#### 多種鍵盤配置

支援標準注音、倚天、許氏、倚天 26 鍵、IBM 以及漢語拼音等六種鍵盤排列方式，選擇蠻豐富的。

<span class="pricing-tag">開源免費</span>
<a href="https://mcbopomofo.openvanilla.org" target="_blank" class="official-link">官方網站</a>

## 效率工具

### Raycast

<Image src={raycast} alt="Raycast" />

我用來取代 Mac 內建 Spotlight 的啟動器工具，幾乎可以把很多日常操作都集中在這裡完成。

#### 剪貼簿管理

內建的剪貼簿歷史功能，可以回溯之前複製過的內容，不用再擔心不小心覆蓋掉剪貼簿裡的東西。

#### Snippets 快捷輸入

可以自訂常用的文字片段，設定觸發關鍵字後就能快速輸入。像是常用的 email 地址、固定回覆的文字模板之類的，打幾個字就能展開，省下不少重複打字的時間。

#### Store 插件生態

Raycast 有自己的 Store，裡面有各式各樣的插件可以裝。我自己比較常用的幾個：

- **Window Management**：Raycast 原生的視窗管理，可快速調整視窗位置跟大小
- **Homebrew**：不用開終端機就能執行安裝跟管理 service
- **Bitwarden**：快速查詢密碼
- **Ente Auth**：直接從 Raycast 搜尋並複製 2FA 驗證碼
- **Calendar**：整合行事曆搜尋，快速查看接下來的行程
- **Easy New File**：快速建立各種格式的文件
- **Google Translate**：快速翻譯
- **Amphetamine**：快速開關防休眠功能
- **TinyPNG**：快速壓縮圖片
- **Visual Studio Code**：快速開啟最近的專案或檔案
- **Tabby**：快速操作 SSH 終端機連線管理
- **Ghostty**：快速操作終端機

<span class="pricing-tag freemium">免費使用，部分功能需訂閱</span>
<a href="https://raycast.com" target="_blank" class="official-link">官方網站</a>

<Image src={rectangle} alt="Rectangle" />

<span class="pricing-tag">開源免費</span>

### Shottr

<Image src={shottr} alt="Shottr" />

我主要用的截圖與標註工具，用原生 Swift 開發，啟動速度極快、記憶體佔用極低，不會拖慢系統。

#### 捲動截圖

支援擷取超出螢幕可視範圍的長頁面內容，自動拼接成完整的長截圖。這是 macOS 內建截圖工具做不到的，遇到需要截整頁的場景非常好用。

#### 像素級測量與取色

提供精確的尺寸測量跟顏色取樣功能，可以直接在截圖上量距離、吸顏色，我在做前端開發或看設計稿的時候會用到。

#### 快速馬賽克

可以一鍵對截圖中的敏感資訊進行模糊處理或馬賽克遮蔽，像是個人資料、密碼之類的，操作非常快速。

#### OCR 文字辨識

內建光學字元辨識功能，可以直接從截圖中擷取文字，不用再另外裝其他工具。

#### 截圖釘選

支援把截圖釘選在螢幕最上層，方便對照參考，不用反覆切換視窗。

<span class="pricing-tag freemium">大部分功能免費，付費可去廣告</span>
<a href="https://shottr.cc" target="_blank" class="official-link">官方網站</a>

### Bob

<Image src={bob} alt="Bob" />

我在 Mac 上用的翻譯跟 OCR 工具，幾乎在任何場景下都能快速完成翻譯，不用離開當前的工作視窗。

#### 劃詞翻譯

選取任意文字後就能快速翻譯，支援幾乎所有 Mac 應用程式。不需要切換視窗、不需要開瀏覽器，選起來就能翻。

#### 截圖翻譯

透過截圖的方式辨識畫面中的文字並翻譯，特別適合翻譯圖片、影片字幕或那些沒辦法直接複製的文字。

#### 多引擎翻譯

支援 Google、DeepL、OpenAI 等多種翻譯引擎，還可以同時顯示多個翻譯結果進行比對，選出最合適的翻譯。

#### OCR 文字辨識

內建離線 OCR 功能，可以從圖片中擷取文字並直接翻譯，處理無法選取的文字內容時非常方便。

#### 輸入翻譯

也支援手動輸入文字進行翻譯，可以當作獨立的翻譯視窗來使用，適合需要主動查詢的情境。

<span class="pricing-tag paid">付費買斷</span>
<a href="https://bobtranslate.com" target="_blank" class="official-link">官方網站</a>

### rclone

<Image src={rclone} alt="rclone" />

一款命令列的雲端儲存管理工具，支援的雲端服務非常多，像是 Google Drive、OneDrive、S3 等等。

## 安全與隱私

### Bitwarden

<Image src={bitwarden} alt="Bitwarden" />

開源的密碼管理工具，用來統一管理所有的密碼跟密鑰（Passkey）。

#### 密碼與 Passkey 管理

主要用來儲存各種網站跟服務的密碼，也支援新一代的 Passkey。搭配瀏覽器擴充套件，自動填入的體驗很順暢，不用再記一堆密碼。

#### 資料夾分類

可以用資料夾來分類管理不同類型的帳號，像是工作、個人、金融等等，帳號一多的時候找起來比較方便。

#### 幾乎免費的成本

Bitwarden 是少數可以用很低成本使用的密碼管理工具，免費方案就已經涵蓋了大部分日常需求，跟其他動輒年費上千的密碼管理工具比起來親民很多。

#### 為什麼不用內建的 2FA？

雖然 Bitwarden 本身也有提供 2FA 驗證碼的功能，但我個人覺得把密碼跟兩步驟驗證碼放在同一個地方不太安全——如果 Bitwarden 被攻破，等於兩道防線同時失守。所以我另外用了 Ente Auth 來獨立管理 2FA 驗證碼。

<span class="pricing-tag">開源免費</span>
<a href="https://bitwarden.com" target="_blank" class="official-link">官方網站</a>
<a href="/articles/password-management" class="official-link">延伸閱讀：密碼與帳號管理</a>

### Ente Auth

<Image src={enteAuth} alt="Ente Auth" />

一款開源且免費的兩步驟驗證（2FA）工具，用來獨立管理各種服務的驗證碼。

#### 開源且免費

Ente Auth 完全開源，而且免費就能使用全部功能，包含端對端加密跟雲端備份。換裝置的時候不用擔心驗證碼遺失，也不用為了基本功能付費。

#### 標籤分類

支援用 Tag 的方式來分類管理驗證碼，像是工作、個人、社群等等，帳號多的時候可以快速篩選，找起來很方便。

#### Raycast 整合

支援 Raycast 的插件系統，可以直接透過 Raycast 快速搜尋並取得驗證碼，複製後直接貼上，不用另外打開 Ente Auth 的視窗，整個流程非常快。

設定流程如下：

- 安裝 Ente 的 CLI 工具並登入帳號。
- 在 Raycast 的 Script Commands 裡新增一個 script，放入 `ente export` 的指令。
- 透過 Raycast 插件的 import 功能讀取匯出的資料，即可在 Raycast 裡直接搜尋驗證碼。

<span class="pricing-tag">開源免費</span>
<a href="https://ente.io/auth" target="_blank" class="official-link">官方網站</a>
<a href="/articles/password-management" class="official-link">延伸閱讀：密碼與帳號管理</a>

### Tailscale

<Image src={tailscale} alt="Tailscale" />

基於 WireGuard 的私有網路工具，可以把不同地點的裝置串成一個安全的虛擬區域網路。

#### 內網服務管理

我自己沒有公網 IP，所以會用 Tailscale 來存取內網的服務。設定好之後，不管人在哪裡都能安全地連回家裡或辦公室的裝置，就像在同一個區域網路下一樣。

#### Windows 遠端連線

搭配 Windows 的遠端桌面連線（RDP）使用，透過 Tailscale 建立的虛擬網路，即使沒有公網 IP 也能順利連上遠端的 Windows 電腦，連線穩定、設定也很簡單。

#### 全平臺支援且免費額度大方

Tailscale 幾乎全平臺都能安裝——Mac、Windows、Linux、iOS、Android 通通支援。而且免費方案的額度非常大方，個人使用基本上不太會碰到限制，可以輕鬆把所有裝置串在一起。

### Zen Browser

<Image src={zen} alt="Zen Browser" />

基於 Firefox 的開源瀏覽器，把 Arc 那套以工作流為核心的 UX 在 Firefox 底層上重新實作一次。

<div class="callout info">
我原本用 [Arc](/articles/arc-workflow) 用了很久，是我目前用過 UX 最好的瀏覽器；後來 Arc 宣布停止主要功能開發，就轉到 [Zen Browser](/articles/zen-browser-setup) 接手。Arc 的工作流幾乎全部都能在 Zen 上延續，是目前看起來最接近 Arc 的接班選項。
</div>

#### 工作區 + 垂直側邊欄

左側垂直側邊欄管理分頁、左右滑動切換工作區，把 Arc 的側邊欄跟 Spaces 原封不動搬過來。每個工作區有自己的 pin 分頁，工作用的常用網站、個人用的各自分開，彼此不混。

#### Split View：原生分割視窗

同一個視窗內把兩個（或更多）分頁並排顯示，不用再開兩個瀏覽器視窗手動排位置。拿來同時對照文件跟程式碼、或一邊看影片一邊查資料都很順。

#### Glance：預覽層開連結

按住觸發鍵（預設 `Ctrl`）再點連結，網頁會用預覽層疊在當前分頁上顯示，看完關掉就好、不會開新分頁。對應到 Arc 的 Peek，查資料不污染分頁這件事完整繼承。

#### Command Bar：鍵盤操作入口

`Cmd+T` 叫出類似 Spotlight 的命令列，搜尋、開新分頁、跳到已開的分頁、執行指令全部從這裡做，鍵盤操作效率很高。

#### 多帳號容器隔離

透過 Firefox 原生的 container 機制做帳號隔離，公司 Gmail 跟個人 Gmail 可以分別登在不同工作區，不用開無痕或裝分身。

### Firefox

<Image src={firefox} alt="Firefox" />

Mozilla 開發的開源瀏覽器，主打隱私保護，是我除了 Zen 之外會搭配使用的瀏覽器，手機端也靠 Firefox Sync 接住 Zen 的歷史紀錄。

#### 強化隱私保護

內建「加強型追蹤保護」功能，預設就會封鎖第三方 Cookie、社群媒體追蹤器跟數位指紋辨識，不用額外裝套件就能有不錯的隱私防護。

#### 豐富的擴充套件

Firefox 的附加元件生態系很成熟，從廣告封鎖到開發者工具應有盡有，可以根據自己的需求自由擴充瀏覽器功能。

#### 容器分頁（Container Tabs）

可以在同一個瀏覽器裡用不同身分隔離分頁，例如把工作、個人、購物分開，各容器的 Cookie 跟登入狀態彼此獨立。這個功能我覺得蠻實用的，不用再為了切換帳號開無痕視窗。

#### 跨平臺同步

透過 Firefox 帳號可以在不同裝置間同步書籤、歷史紀錄、密碼跟開啟的分頁，桌面跟手機之間的瀏覽體驗可以無縫銜接。
