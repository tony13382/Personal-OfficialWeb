## 我怎麼管理擴充功能

在介紹個別擴充功能之前，先聊聊我怎麼管理這些擴充。不同瀏覽器的管理方式其實蠻不一樣的。

### 工作區風格的瀏覽器：直接用原生工作區

<Image src={arc} alt="工作區風格瀏覽器的擴充管理" />

像 Arc、Zen 這類原生支援「工作區」概念的瀏覽器，可以快速切換到不同情境。所以我不會額外裝插件管理工具來做模式切換，而是直接利用工作區來區分不同情境下需要的擴充功能。比方說工作用的工作區開一組擴充，個人瀏覽的工作區開另一組，彼此不會互相干擾。

### Chrome 瀏覽器：搭配 Extension Manager

<Image src={extensionManager} alt="Extension Manager" />

如果是用 Chrome 的話就不太一樣了，因為 Chrome 切換工作區（Profile）本身比較麻煩，每次都要點好幾下。所以我會搭配 Extension Manager 這個擴充來管理，它可以讓你快速啟用或停用一組擴充功能，等於是用比較輕量的方式達到類似工作區切換的效果。

<span class="pricing-tag">免費</span>
<a href="https://chromewebstore.google.com/detail/extension-manager/gjldcdngmdknpinoemndlidpcabkggco" target="_blank" class="official-link">Chrome Web Store</a>

## 帳號與工具

這些是我日常會用到的帳號管理跟小工具，從密碼自動填入到圖片格式轉換，都是讓日常操作更順手的擴充。

### Claude

<Image src={claude} alt="Claude" />

Anthropic 的 Claude AI 擴充，可以在瀏覽器裡直接跟 Claude 對話，不用另外開分頁到 Claude 的網站。在工作中需要快速問問題或整理想法的時候很方便。

<span class="pricing-tag freemium">免費使用，進階功能需訂閱</span>
<a href="https://chromewebstore.google.com/detail/claude/fcoeoabgfenejglbffodgkkbkcdhcgfn" target="_blank" class="official-link">Chrome Web Store</a>

### Save image as Type

<Image src={saveImageAsType} alt="Save image as Type" />

可以在儲存圖片的時候選擇要存成什麼格式，像是 PNG、JPG、WebP 之間互轉，不用再額外用工具做格式轉換。

<span class="pricing-tag">免費</span>
<a href="https://chromewebstore.google.com/detail/save-image-as-type/aninmieokgdbdcgoiglchhhhbpnpolja" target="_blank" class="official-link">Chrome Web Store</a>

### Bitwarden

<Image src={bitwarden} alt="Bitwarden" />

密碼管理擴充，搭配 Bitwarden 主程式使用，可以在瀏覽器上自動填入密碼跟 Passkey，登入各種服務的時候不用再手動輸入。

<span class="pricing-tag">免費</span>
<a href="https://chromewebstore.google.com/detail/bitwarden-password-manage/nngceckbapebfimnlniiiahkandclblb" target="_blank" class="official-link">Chrome Web Store</a>
<a href="https://addons.mozilla.org/en-US/firefox/addon/bitwarden-password-manager/" target="_blank" class="official-link">Firefox Add-ons</a>

### Checker Plus for Gmail

<Image src={checkerPlusGmail} alt="Checker Plus for Gmail" />

可以直接在瀏覽器右上角查看 Gmail 的新郵件，不用另外開 Gmail 的分頁。有新信的時候會即時通知，快速預覽後決定要不要點進去看。

<span class="pricing-tag freemium">免費使用，進階功能需付費</span>
<a href="https://chromewebstore.google.com/detail/checker-plus-for-gmail/oeopbcgkkoapgobdbedcemjljbihmemj" target="_blank" class="official-link">Chrome Web Store</a>
<a href="https://addons.mozilla.org/en-US/firefox/addon/checker-plus-gmail/" target="_blank" class="official-link">Firefox Add-ons</a>

## 瀏覽體驗

日常瀏覽時用來提升體驗的擴充，從廣告封鎖、深色模式到分頁管理，讓瀏覽器用起來更順手。

### uBlock Origin

<Image src={ublockOrigin} alt="uBlock Origin" />

我用的廣告封鎖擴充，開源免費，效能也很好。裝了之後瀏覽體驗乾淨很多，也能減少一些追蹤器。

<span class="pricing-tag">開源免費</span>
<a href="https://chromewebstore.google.com/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm" target="_blank" class="official-link">Chrome Web Store</a>
<a href="https://addons.mozilla.org/en-US/firefox/addon/ublock-origin/" target="_blank" class="official-link">Firefox Add-ons</a>

### SponsorBlock

<Image src={sponsorblock} alt="SponsorBlock" />

看 YouTube 的時候可以自動跳過贊助片段、開頭動畫、結尾推廣等等。資料都是社群貢獻的，用起來省了不少時間。

<span class="pricing-tag">開源免費</span>
<a href="https://chromewebstore.google.com/detail/sponsorblock-for-youtube/mnjggcdmjocbbbhaepdhchncahnbgone" target="_blank" class="official-link">Chrome Web Store</a>
<a href="https://addons.mozilla.org/en-US/firefox/addon/sponsorblock/" target="_blank" class="official-link">Firefox Add-ons</a>

### Return YouTube Dislike

<Image src={returnYoutubeDislike} alt="Return YouTube Dislike" />

把 YouTube 隱藏的倒讚數量顯示回來，在判斷影片品質的時候多一個參考指標。

<span class="pricing-tag">開源免費</span>
<a href="https://chromewebstore.google.com/detail/return-youtube-dislike/gebbhagfogifgggkldgodflihgfeippi" target="_blank" class="official-link">Chrome Web Store</a>
<a href="https://addons.mozilla.org/en-US/firefox/addon/return-youtube-dislikes/" target="_blank" class="official-link">Firefox Add-ons</a>

### Dark Reader

<Image src={darkReader} alt="Dark Reader" />

可以把幾乎所有網頁都轉成深色模式，晚上瀏覽的時候眼睛比較舒服。它會智慧分析網頁的配色去做轉換，大部分網站的效果都還不錯。

<span class="pricing-tag">開源免費</span>
<a href="https://chromewebstore.google.com/detail/dark-reader/eimadpbcbfnmbkopoojfekhnkhdbieeh" target="_blank" class="official-link">Chrome Web Store</a>
<a href="https://addons.mozilla.org/en-US/firefox/addon/darkreader/" target="_blank" class="official-link">Firefox Add-ons</a>

### OneTab

<Image src={onetab} alt="OneTab" />

可以一鍵把目前所有的分頁收成一個列表，釋放記憶體。以前用 Chrome 的時候我很常用它來整理分頁，不過如果你的瀏覽器原生就有工作區或分頁群組功能（例如 Arc、Zen），這個擴充就比較用不太到；但在 Chrome 上還是蠻推薦的。

<span class="pricing-tag">免費</span>
<a href="https://chromewebstore.google.com/detail/onetab/chphlpgkkbolifaimnlloiipkdnihall" target="_blank" class="official-link">Chrome Web Store</a>
<a href="https://addons.mozilla.org/en-US/firefox/addon/onetab/" target="_blank" class="official-link">Firefox Add-ons</a>

### 書籤側邊欄

<Image src={bookmarkSidebar} alt="書籤側邊欄" />

讓書籤用側邊欄的方式顯示，比起瀏覽器內建的書籤管理器更方便存取跟整理。如果你用的瀏覽器本身就有側邊欄（例如 Arc、Zen），它們內建的體驗通常更好，這個擴充就不一定需要。

<span class="pricing-tag">免費</span>
<a href="https://chromewebstore.google.com/detail/bookmark-sidebar/jdbnofccmhefkmjbkkdkfiicjkgofkdh" target="_blank" class="official-link">Chrome Web Store</a>

### QR Code Generator

<Image src={qrCodeGenerator} alt="QR Code Generator" />

可以快速把目前頁面的網址產生成 QR Code，需要在手機上開同一個頁面的時候掃一下就好，不用再手動輸入網址。

<span class="pricing-tag">免費</span>
<a href="https://chromewebstore.google.com/detail/qr-code-generator/afpbjjgbdimpioenaedcjgkaigggcdpp" target="_blank" class="official-link">Chrome Web Store</a>

## 閱讀

閱讀外文內容跟管理知識時用到的擴充，從翻譯、內容過濾到筆記收集，串起整個閱讀工作流。

### 沉浸式翻譯

<Image src={immersiveTranslate} alt="沉浸式翻譯" />

我最常用的網頁翻譯擴充，可以在保留原文的情況下，把翻譯結果以雙語對照的方式顯示在下方。閱讀外文文章的時候非常好用，不會像 Google 翻譯那樣整頁替換掉原文。

<span class="pricing-tag freemium">免費使用，進階功能需訂閱</span>
<a href="https://chromewebstore.google.com/detail/immersive-translate-web-p/bpoadfkcbjbfhfodiogcnhhhpibjhbnh" target="_blank" class="official-link">Chrome Web Store</a>
<a href="https://addons.mozilla.org/en-US/firefox/addon/immersive-translate/" target="_blank" class="official-link">Firefox Add-ons</a>

### GlotDojo

<Image src={glotdojo} alt="GlotDojo" />

即時翻譯字幕的擴充，看外語影片的時候可以同時顯示原文跟翻譯字幕，方便學習外語或理解內容。

<span class="pricing-tag freemium">免費使用，進階功能需付費</span>
<a href="https://chromewebstore.google.com/detail/glot-extension/dbnjpielondlkmdjbembloegkaabfakc" target="_blank" class="official-link">Chrome Web Store</a>

### 為什麼你們就是不能加個空格呢？

<Image src={panguSpacing} alt="為什麼你們就是不能加個空格呢？" />

自動在中英文之間加上空格，讓網頁上的文字排版更好看、更好閱讀。裝了之後看中英混排的內容會舒服很多。

<span class="pricing-tag">開源免費</span>
<a href="https://chromewebstore.google.com/detail/%E7%82%BA%E4%BB%80%E9%BA%BC%E4%BD%A0%E5%80%91%E5%B0%B1%E6%98%AF%E4%B8%8D%E8%83%BD%E5%8A%A0%E5%80%8B%E7%A9%BA%E6%A0%BC%E5%91%A2%EF%BC%9F/paphcfdffjnbcgkokihcdjliihicmbpd" target="_blank" class="official-link">Chrome Web Store</a>

<a href="https://addons.mozilla.org/en-US/firefox/addon/content-farm-terminator/" target="_blank" class="official-link">Firefox Add-ons</a>

### Save to Notion

<Image src={saveToNotion} alt="Save to Notion" />

可以把網頁內容快速儲存到 Notion 的資料庫裡，我會用它來收集有參考價值的文章跟資料，方便之後回頭查閱。

<span class="pricing-tag">免費</span>

<Image src={notionWebClipper} alt="Notion Web Clipper" />

<span class="pricing-tag">免費</span>
