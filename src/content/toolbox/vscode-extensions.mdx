## 外觀與介面

### Material Icon Theme

<Image src={materialIconTheme} alt="Material Icon Theme" style="max-width: 96px;" />

讓 VS Code 的檔案圖示更好辨識，不同類型的檔案跟資料夾都有對應的 Material Design 風格圖示，在專案結構複雜的時候可以更快找到目標檔案。

<span class="pricing-tag">免費</span>
<a href="https://marketplace.visualstudio.com/items?itemName=PKief.material-icon-theme" target="_blank" class="official-link">VS Code Marketplace</a>

### One Dark Pro

<Image src={oneDarkPro} alt="One Dark Pro" style="max-width: 96px;" />

我在用的佈景主題，配色舒適、對比適中，長時間看程式碼不容易疲勞。是從 Atom 編輯器經典的 One Dark 主題移植過來的。搭配的字體是 Roboto Mono，這是我自己用網站測試過各種等寬字體後，覺得視覺效果最舒服的選擇。如果你也想找到最適合自己的字體，可以試試 <a href="https://devfonts.gafi.dev/" target="_blank">Dev Fonts</a> 跟 <a href="https://www.codingfont.com/" target="_blank">Coding Font</a> 這兩個網站來比較。

<span class="pricing-tag">免費</span>
<a href="https://marketplace.visualstudio.com/items?itemName=zhuangtongfa.Material-theme" target="_blank" class="official-link">VS Code Marketplace</a>

### Chinese (Traditional) Language Pack

<Image src={zhHant} alt="繁體中文語言包" style="max-width: 96px;" />

VS Code 的繁體中文語言包，把介面翻譯成中文，降低操作上的語言門檻。

<span class="pricing-tag">免費</span>
<a href="https://marketplace.visualstudio.com/items?itemName=MS-CEINTL.vscode-language-pack-zh-hant" target="_blank" class="official-link">VS Code Marketplace</a>

### Indent Rainbow

<Image src={indentRainbow} alt="Indent Rainbow" style="max-width: 96px;" />

用不同顏色標示程式碼的縮排層級，在巢狀結構比較深的時候可以更快辨識目前在哪一層，減少縮排錯誤。

<span class="pricing-tag">免費</span>
<a href="https://marketplace.visualstudio.com/items?itemName=oderwat.indent-rainbow" target="_blank" class="official-link">VS Code Marketplace</a>

## 通用

### GitLens

<Image src={gitlens} alt="GitLens" style="max-width: 96px;" />

Git 增強擴充，可以在程式碼旁邊直接看到每一行是誰在什麼時候改的（blame），也能快速查看檔案的完整修改歷史，在追蹤問題來源的時候很實用。

<span class="pricing-tag freemium">免費使用，進階功能需訂閱</span>
<a href="https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens" target="_blank" class="official-link">VS Code Marketplace</a>

### Git Graph

<Image src={gitGraph} alt="Git Graph" style="max-width: 96px;" />

用圖形化的方式呈現 Git 分支的歷史，可以清楚看到分支的合併、分叉跟提交紀錄。在處理比較複雜的分支流程時，比純命令列直覺很多。

<span class="pricing-tag">免費</span>
<a href="https://marketplace.visualstudio.com/items?itemName=mhutchie.git-graph" target="_blank" class="official-link">VS Code Marketplace</a>

### Path Intellisense

<Image src={pathIntellisense} alt="Path Intellisense" style="max-width: 96px;" />

在輸入檔案路徑時會自動補全，不用記住完整的路徑或手動翻資料夾結構，import 檔案的時候特別方便。

<span class="pricing-tag">免費</span>
<a href="https://marketplace.visualstudio.com/items?itemName=christian-kohler.path-intellisense" target="_blank" class="official-link">VS Code Marketplace</a>

### Todo Tree

<Image src={todoTree} alt="Todo Tree" style="max-width: 96px;" />

會掃描專案裡所有的 TODO、FIXME 等標記，用樹狀結構整理在側邊欄裡，方便追蹤還沒處理完的待辦事項，不會遺漏。

<span class="pricing-tag">免費</span>
<a href="https://marketplace.visualstudio.com/items?itemName=Gruntfuggly.todo-tree" target="_blank" class="official-link">VS Code Marketplace</a>

### Bookmarks

<Image src={bookmarks} alt="Bookmarks" style="max-width: 96px;" />

可以在程式碼裡標記書籤，快速跳轉到標記的位置。在閱讀大型專案或需要頻繁切換不同段落時很好用。

<span class="pricing-tag">免費</span>
<a href="https://marketplace.visualstudio.com/items?itemName=alefragnani.bookmarks" target="_blank" class="official-link">VS Code Marketplace</a>

### filesize

<Image src={filesize} alt="filesize" style="max-width: 96px;" />

在狀態列顯示目前開啟檔案的大小，對於需要注意檔案體積的場景（像是圖片、打包後的檔案）很實用。

<span class="pricing-tag">免費</span>
<a href="https://marketplace.visualstudio.com/items?itemName=mkxml.vscode-filesize" target="_blank" class="official-link">VS Code Marketplace</a>

### Remote - SSH

<Image src={remoteSsh} alt="Remote - SSH" style="max-width: 96px;" />

透過 SSH 連線到遠端伺服器，直接在 VS Code 裡編輯遠端的檔案，操作體驗跟在本地開發幾乎一樣。

<span class="pricing-tag">免費</span>
<a href="https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-ssh" target="_blank" class="official-link">VS Code Marketplace</a>

### Draw.io Integration

<Image src={drawio} alt="Draw.io Integration" style="max-width: 96px;" />

直接在 VS Code 裡開啟跟編輯 Draw.io 的圖表，不用切換到瀏覽器。畫流程圖、架構圖的時候可以跟程式碼放在一起管理。

<span class="pricing-tag">免費</span>
<a href="https://marketplace.visualstudio.com/items?itemName=hediet.vscode-drawio" target="_blank" class="official-link">VS Code Marketplace</a>

### Markdown PDF

<Image src={markdownPdf} alt="Markdown PDF" style="max-width: 96px;" />

可以直接在 VS Code 裡把 Markdown 檔案轉成 PDF，算是目前做 Markdown 轉 PDF 效果最好的套件。排版跟格式的還原度很高，需要產出文件的時候非常方便。

<span class="pricing-tag">免費</span>
